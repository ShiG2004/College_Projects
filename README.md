# College Projects
Information system (Cybersecurity) projects at University of Pittsburgh at Johnstown. It has a collection of hands-on college projects covering cybersecurity, cloud computing, networking, system hardening, and IT infrastructure, with practical experience in AWS, Azure, Windows security, and network defense.

### AWS Firewalling Project: 
Description: Configured AWS security groups and VPC rules to control inbound and outbound network traffic and test access restrictions.<br>
Tools: AWS, VPC, Security Groups, EC2

### AWS IAM RBAC:
Description: Implemented role-based access control using AWS IAM to enforce least-privilege permissions.<br>
Tools: AWS IAM, AWS Console

### Using Azure IaaS & PaaS:
Desciption: Deployed cloud resources using both Azure IaaS and PaaS models to compare scalability and management responsibilities.<br>
Tools: Microsoft Azure, Azure VMs, Azure App Services
